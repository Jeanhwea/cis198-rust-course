* 学习背景
1. rust 和 c/c++/Java/Python/golang 不太一样
2. rust 学习曲线比较陡峭
3. rust 有很多颠覆认知的特性：
   - 所有权，生命周期，借用检测
   - cargo 工具
   - 函数式+命令式支持
4. 视频讲解见 [[https://space.bilibili.com/1969478249/channel/collectiondetail?sid=2542770][B站]]
