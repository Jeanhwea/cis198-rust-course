* unsafe
** safe vs unsafe
1. safe 保证的
   - Ownership
   - Type Checking
   - Memory safety rules
2. unsafe 不能保证的
   - Deadlock
   - Memory leak
   - Integer overflow
   - 退出是忘记调用 destructor
** unsafe 函数
1. unsafe block
2. unsafe function
